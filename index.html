Non-Academic Projects

  • Talking Hands
     Talking Hands is a desktop application that allows people with speaking disabilities to communicate virtually using American Sign Language (ASL). The application uses machine learning to translate ASL into English speech through video feeds. I worked on collecting data sets, training the Microsoft Azure Custom Vision model, and implementing the TensorFlow model into the test Android application. As a result of our efforts, my team was declared the as winner of the Google Cloud Product COVID-19 Hackathon Fund.

    View Project 🡕

  • The Dazed Maze
     The Dazed Maze is a text-based maze game where you start in a room and you have to navigate through other rooms to get to the correct room. The game uses the concepts of Object-Oriented programming and Data Structures to function.

    View Project 🡕

  • Protect the Paywall
     Protect the Paywall is a JavaScript algorithm that detects security flaws in major newspaper sites regarding paywalls on web pages. The script is tested to work on sites including but not limited to The New York Times, Los Angeles Times, The Atlantic, and The Economist. I have also notified the site engineers of the listed websites about the vulnerability.

    View Project 🡕

     The UT Directory Crawler is a robust web scraper that bulk extracts public information about UT-affiliated members through the UT Directory website. The scraper uses the BeautifulSoup4 Python library to scrape and organize data into CSV/JSON formats.
